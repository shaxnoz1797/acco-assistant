# AccoAssistant - Modern Accounting Management Dashboard

AccoAssistant is a streamlined web application designed for accountants to manage multiple business entities, monitor their Tax Identification Numbers (TIN/STIR), and track VAT (Value Added Tax) statuses in real-time.

## 🚀 Key Features
- **Firm Management:** View all registered firms in a clean, card-based layout.
- **Search & Filtering:** Quickly find firms by name or Tax ID (TIN).
- **CRUD Operations:** Easily add new firms, edit existing details, or remove them from the list.
- **VAT Status Tracking:** Visual indicators to quickly identify VAT-registered entities.
- **Responsive Design:** Fully functional on various screen sizes, from desktops to tablets.
- **Pagination:** Smooth navigation through large sets of data.

## 🛠 Tech Stack
This project is built using modern technologies:
- **Frontend:** [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/) (or any icon library you used)
- **Backend:** [Python](https://www.python.org/) + [Django](https://www.djangoproject.com/) / [FastAPI](https://fastapi.tiangolo.com/)
- **Database:** SQLite
